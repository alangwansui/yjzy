Advance Global Search

This module enable you to global search feature, you can search any records that you have access and configured relavent models user wise

List of Features


  • It Work With Both Community And Enterprise Version.
  • Search with a shortcut (Be anywhere in odoo and a shortcut will get you to search area).
  • Easy To Configur & Search User Wise.
  • You can search any records that you have access and configured relavent models.
  • Don't need to go to individual object(like sales, purchase, invoices etc..) to search records.
  • Find all objects record at one click at one place.
  • Global Search configuration can only see the "Manager", "User" can only see his/her own record.
  • User & Manager can search record from anywhere using click on search icon or press toggle (Alt + Q) Shortcut.

Setup & Configuration

As shown below set Global search "Manager" To see the confuguration & Global search Menu.


For configur Go to the Global Searche >> Global Searches >> Global Searches Select the user, model & fields from you want to search.


How it works

You can search any records with toggle shortcut (Alt + Q), Type in search box then result will be display from result select record that you want to open
