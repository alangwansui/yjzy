Select and delete multiple records simultaneously from X2M in the Form view.

If you have multiple records of O2M or M2M fields to be deleted in the Form view, you need to delete them one by one. Unlike List view, you cannot select the multiple records at a time and delete them simultaneously. We have solved your this problem!.

This module is developed for deleting your records containing One2Many(O2M) or Many2Many (M2M) fields in the Form View.

List of Features


  • It Work With Both Community And Enterprise Version.
  • You just need to select the records to be deleted containing O2M or M2M fields and then click on delete icon.
  • On click on the uppermost delete icon in the list. A pop-up will appears asking for confirmation to delete all the records in One2Many (O2M) and Many2Many (M2M) fields.
  • If user have not selected any records then user click on uppermost delete icon then warning message will be appear for select at least one record.

How it works

Delete multiple records in One2Many (O2M) fields.


Delete multiple records in Many2Many (M2M) fields.
